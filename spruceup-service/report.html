Report

  1. This service helps name and keep track of plants.
  2. Uses MySQL database.
  3. Provides an Express-based web service.
  4. Available on https://project2.nickc.me:8443/
  5. Endpoints and parameters are named appropriately.
  6. Send and receives data as JSON.
  7. Create, read, update, and delete (CRUD) operations are supported.
  8. Uses ufw to block all unused ports.
  9. Created Nginx server for global, encrypted access to the web service.
  10. Supports cross-origin resource sharing and an options wildcard endpoint.
  11. Web service was started using pm2 to stay running.
  12. Stored in a Git repository shared with twodee.
  13. Database's creation and schema scripts in git repository.
  14. Could have set up git repository earlier.
  15. See report here.
Here are some examples of how I used curl to test the site.

Read 

    curl https://project2.nickc.me:8443/plants/firebreather

Create 

    curl --header 'Content-Type: application/json' \
    --data '{"nickname": "puffle", "type": "Gymnosporangium Juniperi Virginianae",
    "description": "Orange", "height": 5}' \
    https://project2.nickc.me:8443/plants

    curl https://project2.nickc.me:8443/plants/puffle

Update 

    curl --header 'Content-Type: application/json' \
    --request PATCH \
    --data '{"nickname": "day-zee", "type": "daisy",
    "description": "Orange", "height": 2918}' \
    https://project2.nickc.me:8443/plants/8

    curl https://project2.nickc.me:8443/plants/day-zee

Delete 

    curl --request DELETE \
    https://project2.nickc.me:8443/plants/2
